# Coverage

This document describes the data coverage of the Italian Data Protection MCP server.

## Authority

**Garante per la protezione dei dati personali** (Garante)
Italian supervisory authority for data protection under GDPR.
Official website: https://www.garanteprivacy.it/

## Decisions

| Type | Description |
|---|---|
| `sanzione` | Administrative fines issued against controllers/processors |
| `provvedimento` | Prescriptions, orders, and corrective measures |
| `ordinanza` | Injunction orders (ordinanze-ingiunzione) |
| `parere` | Opinions and advisory decisions |

## Guidelines

| Type | Description |
|---|---|
| `linee_guida` | Official guidelines on GDPR implementation topics |
| `provvedimento_generale` | General provisions applicable to sectors or processing types |
| `FAQ` | Frequently asked questions on data protection obligations |
| `parere` | Advisory opinions on legislative proposals and sectoral rules |

## Language

All decisions and guidelines are primarily in **Italian**. Some documents may have an English translation or summary.

## Update frequency

Data is refreshed by the scheduled `ingest.yml` workflow. Check `it_dp_check_data_freshness` for the most recent record dates.
